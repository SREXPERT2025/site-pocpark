### Задача

Организовать безопасный и удобный проезд для жителей, не превращая двор в проходной двор для всех подряд.

### Решение

ANPR, правила доступа по ролям и события для контроля. Для чувствительных сценариев — двухфакторная идентификация (по политике объекта). Гости оформляются по правилам/заявкам.

### Результат

Удобный въезд для жителей, управляемые гостевые сценарии и прозрачность по событиям для УК и службы безопасности.
