# CTA Block — стандарт вызова действия (РОСПАРК)

## Назначение
`CtaBlock` — единый компонент CTA для всех контентных страниц:
- Решения
- Для руководства
- Для IT-директора
- Кейсы

Используется в `.mdx` файлах.

---

## Установка

1. Скопировать файл:
```
components/ui/CtaBlock.tsx
```

2. Использовать в MDX:
```mdx
import { CtaBlock } from "@/components/ui/CtaBlock";
```

---

## Пример использования

```mdx
<CtaBlock
  title="Готовы рассчитать окупаемость для вашего объекта?"
  description="Подготовим коммерческое предложение с расчетом сроков окупаемости."
  buttonText="Получить расчет под ключ"
  href="/quiz"
  badge="Бесплатно"
/>
```

---

## Рекомендации

- Использовать **не более 2 CTA на страницу**
- Первый — консультация
- Второй — расчёт / КП
- Не менять стиль без согласования (это часть дизайн-системы)
